# Images pour les projets

## RjWeb - Images à ajouter

Placez ici vos captures d'écran PNG du projet RjWeb avec les noms suivants :

- `rjweb-preview.png` - Image principale (utilisée comme vignette)
- `rjweb-utilisateur.png` - Page utilisateur/accueil
- `rjweb-connexion.png` - Page de connexion
- `rjweb-contact.png` - Page de contact
- `rjweb-inscription.png` - Formulaire d'inscription
- `rjweb-inscrit.png` - Liste des inscrits
- `rjweb-profil.png` - Page de profil
- `rjweb-about.png` - Page à propos

## Beroli

Pour le projet Beroli, une image Unsplash est utilisée temporairement.
Si vous avez des captures d'écran de l'application mobile, vous pouvez :
- Créer `beroli-preview.png` (et mettre à jour le chemin dans Projects.astro)

## Format recommandé

- **Format** : PNG ou JPG
- **Largeur recommandée** : 800-1200px
- **Ratio** : 16:9 (paysage) pour une bonne compatibilité
- **Optimisation** : Utilisez TinyPNG ou Squoosh avant de les ajouter
